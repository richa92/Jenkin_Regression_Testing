*** Settings ***
Documentation		Validating hardware utilization temperature
Resource			../resource.txt
Test Setup			Load Test Data and Open Browser
Test Teardown		Logout and close all browsers


*** Test Cases ***
Check for server utilization panel
	${blnUtilizationPanel} =	Set Variable	False
	Log into Fusion appliance as Administrator
	${blnUtilizationPanel} =  Fusion UI Validate Server Utilization Panel		${TestData.server_details}
	Set Global Variable		${blnUtilizationPanel}
	Run Keyword If		'${blnUtilizationPanel}' == '${blnExpected}'	Log		Given server is having utilization panel
	Run Keyword Unless		'${blnUtilizationPanel}' == '${blnExpected}'	Fail	Given sever doesn't have utilization panel

Validate Severer hardware utilization temperature
	Run Keyword Unless		'${blnUtilizationPanel}' == '${blnExpected}'	Fail	Given sever doesn't have utilization panel
	Log into Fusion appliance as Administrator
	${blnValidateTemp} =	Fusion UI Validate Server Temperature		@{TestData.hw_details}
	Run Keyword If		'${blnValidateTemp}' == '${blnExpected}'	Log		Temperature values displayed in appliance and OA are same
	Run Keyword Unless		'${blnValidateTemp}' == '${blnExpected}'	Fail	Temperature values displayed in appliance and OA are not same
