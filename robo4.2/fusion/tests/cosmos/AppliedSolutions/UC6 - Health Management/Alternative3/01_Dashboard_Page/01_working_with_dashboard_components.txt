*** Settings ***
Documentation		Delete Dashboard Component
Resource	        ../resource.txt
Suite Setup         Load Test Data and Open Browser
Suite Teardown      Logout and close all browsers


*** Test Cases ***
Create dash board component
	Log into Fusion appliance as Administrator
	${blnCreateComp} =	Set Variable	False
	${blnCreateComp} =	Fusion UI Create Dashboard		@{TestData.dashboards}
	Set Global Variable		${blnCreateComp}
	Run Keyword If		'${blnCreateComp}' == '${blnExpected}'    Log		Given dash board component is created successfully
	Run Keyword Unless		'${blnCreateComp}' == '${blnExpected}'    Fail	Failed to create given dash board component
	
Verify Dash board component existence after creating
	Run Keyword Unless		'${blnCreateComp}' == '${blnExpected}'    Fail	Failed to create given dash board component
	${blnComponet} =	Set Variable	False
	${blnComponet} =	Fusion UI Dashboard Should Exist		@{TestData.dashboards}
	Set Global Variable		${blnComponet}
	Run Keyword If		'${blnComponet}' == '${blnExpected}'    Log		Given dash board component is present in appliance
	Run Keyword Unless		'${blnComponet}' == '${blnExpected}'    Fail	Given dash board component is not present in appliance

Update Dashboard component
	Run Keyword Unless		'${blnComponet}' == '${blnExpected}'    Fail	Given dash board component is not present in appliance
	${blnUpdateComp} =	Fusion UI Edit Dashboard	@{TestData.editdashboards}
	Run Keyword If	'${blnUpdateComp}' == '{blnExpected}'	Log		Dashboard component updated successfully
	Run Keyword Unless	'${blnUpdateComp}' == '${blnExpected}'	Fail	Failed to update Dashboard component

Delete Dashboard component
	Run Keyword Unless		'${blnComponet}' == '${blnExpected}'    Fail	Given dash board component is not present in appliance
	${blnDeleteComp} =	Fusion UI Delete Dashboard				@{TestData.dashboards}
	Run Keyword If	'${blnDeleteComp}' == '${blnExpected}'    Log	Given dash board component is deleted successfully
	Run Keyword Unless	'${blnDeleteComp}' == '${blnExpected}' 	Fail	Failed to delete dashboard component
