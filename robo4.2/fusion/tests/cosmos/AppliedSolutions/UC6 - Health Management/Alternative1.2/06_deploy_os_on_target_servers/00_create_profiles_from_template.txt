Create profile from Profile Template
	Log into Fusion appliance as Administrator
	${profiles} =	Set Variable	@{TestData.profile_from_template}
	${profile_count} =	Get Length	${profiles}
	:FOR	${i}	IN RANGE	1	${profile_count}
	\	${blnCreateProf} =	Fusion UI Create Profile From Template	${profiles[${i}]}
	\	Run Keyword Unless		'${blnCreateProf}' == 'True'	Fail	Failed to create some/all profiles from profile template.
