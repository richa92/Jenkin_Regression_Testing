Documentation		Create Network sets

Test Setup			Load Test Data and Open Browser

*** Test Cases ***
As an AD user I want to add network sets
	Log into Fusion appliance as Administrator
	${blnactual} =	Fusion UI create network set	@{TestData.networksets}
	Run Keyword If		'${blnactual}' == '${blnExpected}'		Log		Network sets are successfully created.
	Run Keyword Unless		'${blnactual}' == '${blnExpected}'	Fail	Failed to create Network sets

Create label for network sets
	Log into Fusion appliance as Administrator
	${Add_networkset_label} =	Fusion UI add label to network set	@{TestData.networksetlabels}
	Run Keyword If		'${Add_networkset_label}' == '${blnExpected}'		Log		Labels are added to all network sets successfully.
	Run Keyword Unless		'${Add_networkset_label}' == '${blnExpected}'	Fail	Failed to add labels to Some/all of network sets.
