*** Settings ***
Documentation		Create FC and Ethernet networks
Resource			../../resource.txt
Test Setup			Load Test Data and Open Browser
Test Teardown		Logout and close all browsers


*** Test Cases ***
Create FC Networks:
	Log into Fusion appliance as Administrator
	${blnAddFCNet} =	Fusion UI create fc network		@{TestData.fcnetworks}
	Run Keyword If		'${blnAddFCNet}' == '${blnExpected}'		Log		All FC Nets are added successfully.
	Run Keyword Unless		'${blnAddFCNet}' == '${blnExpected}'	Fail	Failed to add Some/all of FC Nets
	
Create Ethernet Networks:
	Log into Fusion appliance as Administrator
	${blnAddEthNet} =	Fusion UI create ethernet network		@{TestData.networks}
	Run Keyword If		'${blnAddEthNet}' == '${blnExpected}'		Log		All Ethernets are added successfully.
	Run Keyword Unless		'${blnAddEthNet}' == '${blnExpected}'	Fail	Failed to add Some/all of Ethernets

Create label for networks
	Log into Fusion appliance as Administrator
	${Add_network_label} =	Fusion UI add label to networks	@{TestData.networklabels}
	Run Keyword If		'${Add_network_label}' == '${blnExpected}'		Log		Labels are added to all networks successfully.
	Run Keyword Unless		'${Add_network_label}' == '${blnExpected}'	Fail	Failed to add labels to Some/all of networks.
