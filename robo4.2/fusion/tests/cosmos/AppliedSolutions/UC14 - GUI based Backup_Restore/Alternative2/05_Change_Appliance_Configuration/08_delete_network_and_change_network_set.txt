Documentation		Delete Network and change Network set Config
Resource			../resource.txt
Test Setup			Load Test Data and Open Browser
Test Teardown		Logout and close all browsers

Library				BuiltIn


*** Test Cases ***
Delete the Network
	Log into Fusion appliance as Administrator
	${bln_delete_network} =	Fusion UI Delete FC Network	${TestData.fcnetworks[2]}
	Run Keyword If		'${bln_delete_network}' == '${blnExpected}'		Log		Network is deleted successfully.
	Run Keyword Unless		'${bln_delete_network}' == '${blnExpected}'	Fail	Failed to delete network.

Change Network Set Configuration
	Log into Fusion appliance as Administrator
	Fusion UI navigate to network sets page
	wait for element and click	${FusionNetworkSetsPage.ID_ACTION_MAIN_BTN}
	wait for element and click	${FusionNetworkSetsPage.ID_MENU_ACTION_EDIT_NETWORK_SET}
	wait for element	${FusionNetworkSetsPage.ID_BTN_EDIT_NETWORKS_NAME, 5}
	wait for element and input text	${FusionNetworkSetsPage.ID_BTN_EDIT_NETWORKS_NAME}	${TestData.networksets[0].new_name}
	wait for element and click	${FusionNetworkSetsPage.ID_BTN_UPDATE_CONFORM}
	Sleep	5
	${bln_newnetworkset_exist} =	Fusion UI select network set	${TestData.networksets[0].new_name}
	Run Keyword If		'${bln_newnetworkset_exist}' == '${blnExpected}'		Log		Changed network set name successfully.
	Run Keyword Unless		'${bln_newnetworkset_exist}' == '${blnExpected}'	Fail	Failed to change network set name.
