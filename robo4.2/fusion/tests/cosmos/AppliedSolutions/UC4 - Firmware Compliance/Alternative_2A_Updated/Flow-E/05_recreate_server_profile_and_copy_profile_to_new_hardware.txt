*** Settings ***
Documentation	Recreate profile and copy profile to new hardware
Resource		../resource.txt
Test Setup		Load Test Data and Open Browser
Test Teardown	Logout and close all browsers

*** Test Cases ***
Recreate server profile
	Log into Fusion appliance as Administrator
	${blnCreateProf} =  Fusion UI create Server Profile		@{TestData.profile_1}
	Run Keyword If		'${blnCreateProf}' == '${blnExpected}'		Log		All Profiles are created successfully.
	Run Keyword Unless		'${blnCreateProf}' == '${blnExpected}'	Fail	Failed to create Some/all of the Profiles mentioned in data file.

Copy profile created on the previous test to new hardware
	Log into Fusion appliance as Administrator
	${blnCreateProf3} =  Fusion UI Copy Server Profile		@{TestData.copyprofiles}
	Run Keyword If		'${blnCreateProf3}' == '${blnExpected}'		Log		Profile successfully copied.
	Run Keyword Unless		'${blnCreateProf3}' == '${blnExpected}'	Fail	Failed to copy Profile.
