*** Test Cases ***
Create FC Networks:
	Login as a Infrastructure administrator of AD user
	${blnactual} =	Fusion UI create fc network		@{TestData.fcnetworks}
	Run Keyword If		'${blnactual}' == '${blnExpected}'    Log		FC networks are successfully added
	Run Keyword Unless		'${blnactual}' == '${blnExpected}'    Fail	Failed to add FC networks

Create Ethernet Networks:
	Login as a Infrastructure administrator of AD user
	${blnactual} =	Fusion UI create ethernet network		@{TestData.networks}
	Run Keyword If		'${blnactual}' == '${blnExpected}'    Log		Ethernet Networks are successfully added
	Run Keyword Unless		'${blnactual}' == '${blnExpected}'    Fail	Failed to add Ethernet Networks
