*** Settings ***
Documentation		Create LIG and EG
Resource 	        ../resource.txt
Test Setup          Load Test Data and Open Browser
Test Teardown       Logout and close all browsers


*** Test Cases ***
Create Logical Interconnect Group:
	Login as a Infrastructure administrator of AD user
 	${blnCreateLIG} =  Fusion UI create logical interconnect group		@{TestData.ligs}
 	Set Global Variable		${blnCreateLIG}
	Run Keyword If		'${blnCreateLIG}' == '${blnExpected}'    Log		All LIGs are added successfully.
	Run Keyword Unless		'${blnCreateLIG}' == '${blnExpected}'    Fail	Failed to add Some/all of the LIGs mentioned in data file.	
	
Create Enclosure group:
    Run Keyword Unless		'${blnCreateLIG}' == '${blnExpected}'    Fail	Failed to add Some/all of the LIGs mentioned in data file.
    Login as a Infrastructure administrator of AD user
    ${blnCreateEG} =  Fusion UI create enclosure group	@{TestData.encgroups}
    Run Keyword If		'${blnCreateEG}' == '${blnExpected}'    Log		All the Enclosure groups are added successfully
	Run Keyword Unless		'${blnCreateEG}' == '${blnExpected}'    Fail		Failed to add Some/all of the Enclosure groups mentioned in data file
