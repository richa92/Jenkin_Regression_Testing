Create FC Networks:
	[Tags]  fcnetwork
	Log into Fusion appliance as Administrator
	${blnFC} =	Fusion UI create fc network		@{TestData.fcnetworks}
	Set Global Variable		${blnFC}
	Run Keyword If			'${blnFC}' == '${blnExpected}'	Log		All FC networks are created successfully.
	Run Keyword Unless		'${blnFC}' == '${blnExpected}'	Fail	Failed to create Some/all of the FC networks mentioned in data file.
