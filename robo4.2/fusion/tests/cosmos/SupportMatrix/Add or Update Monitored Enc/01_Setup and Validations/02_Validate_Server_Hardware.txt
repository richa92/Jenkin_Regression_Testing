Check Server Hardware Page:
	[Documentation]	 Verify Ports, Hardware, Alerts and Utilization information, selecting from the drop down menu
	Log into Fusion appliance as Administrator
	${blnValidateServ} =	Fusion UI Validate Server hardware page		@{TestData.add_enclosure}
	Set Global Variable		${blnValidateServ}
	Run Keyword If		'${blnValidateServ}' == '${blnExpected}'    Log		Server hardware page validated successfully.
	Run Keyword Unless		'${blnValidateServ}' == '${blnExpected}'    Fail	Failed to validate some/all server hardware.

    Log into Fusion appliance as Administrator
	${blnServerOn} =	Fusion UI Power on All Servers
	Set Global Variable		${blnServerOn}
