TC 01: Validate The Enclosure Health Status When A Fan From Bays 1-5 Is Removed And Re-inserted Into An Enclosure
    [Documentation]    This test verifies if an enclosure goes to a Warning state when a fan is removed from bays 1-5
    ...    and returns to OK state when re-inserted back to enclosure
    [Tags]    PE    API    Automated    Fan    EM
    ${present_fan_list}    ${absent_fan_list}=    Get Device Bay Present And Absent List    ${FANDevice_EM}
    ${initial_len}=    Get Length    ${present_fan_list}
    Should Be Equal as Strings    ${initial_len}    10    msg= Enclosure does not have sufficient fans
    ${fan_list}=    Get Slice From List    ${present_fan_list}    0    5
    : FOR    ${fan_bay}    IN    @{fan_list}
    \    #Retrieve the Enclosure health status before efuse-on
    \    ${resp_before_efuseon}=    Fusion Api Get Enclosures
    \    ${health_before_efuseon}=    Get From Dictionary    ${resp_before_efuseon['members'][0]}    status
    \    Should Be Equal    ${health_before_efuseon}    OK
    \    #Efuse-On each fan from bays 1-5 and wait for the fan absence state
    \    Efuse Fan Device   EFuseOn    ${fan_bay}
    \    Wait For Device Present Or Absent Status    ${FANDevice_EM}    ${fan_bay}    Absent
    \    #Check the Enclosure health status after efuse-on
    \    ${resp_after_efuseon}=    Fusion Api Get Enclosures
    \    ${health_after_efuseon}=    Get From Dictionary    ${resp_after_efuseon['members'][0]}    status
    \    Should Be Equal    ${health_after_efuseon}    Warning
    \    #Efuse-Off each fan from bays 1-5 and wait for the fan to return to OK status
    \    Efuse Fan Device    EFuseOff    ${fan_bay}
    \    Wait For Device Status    ${FANDevice_EM}    ${fan_bay}    OK
    \    #Check the Enclosure health status after efuse-off
    \    ${resp_after_efuseoff}=    Fusion Api Get Enclosures
    \    ${health_after_efuseoff}=    Get From Dictionary    ${resp_after_efuseoff['members'][0]}    status
    \    Should Be Equal    ${health_after_efuseoff}    OK
    [Teardown]    Run Keyword If Test Failed    Run Keywords    Efuse Fan Device    EFuseOff    ${fan_bay}    AND    Wait For Device Status    ${FANDevice_EM}    ${fan_bay}    OK

TC 03: Validate The Enclosure Health Status When A Fan Is Removed From Bays 1-5 And Bays 6-10 And Re-inserted Into An Enclosure
    [Documentation]    This test verifies if an enclosure goes to a Warning state when a fan is removed from bays 1-5 and
    ...    from bays 6-10 and returns to OK state when re-inserted back to enclosure
    [Tags]    PE    API    Automated    Fan    EM
    ${present_fan_list}    ${absent_fan_list}=    Get Device Bay Present And Absent List    ${FANDevice_EM}
    ${initial_len}=    Get Length    ${present_fan_list}
    Should Be Equal as Strings    ${initial_len}    10    msg= Enclosure does not have sufficient fans
    #Retrieve the Enclosure health status before efuse-on
    ${resp_before_efuseon}=    Fusion Api Get Enclosures
    ${health_before_efuseon}=    Get From Dictionary    ${resp_before_efuseon['members'][0]}    status
    Should Be Equal    ${health_before_efuseon}    OK
    #Retrieve a random fan bay number from bays 1-5 and from bays 6-10
    ${fan_bay1}=    Evaluate    random.randint(1,5)   random
    ${fan_bay2}=    Evaluate    random.randint(6,10)   random
    #Efuse-On the fan bays and wait for the fan absence state
    Efuse Fan Device   EFuseOn    ${fan_bay1}
    Wait For Device Present Or Absent Status    ${FANDevice_EM}    ${fan_bay1}    Absent
    Efuse Fan Device   EFuseOn    ${fan_bay2}
    Wait For Device Present Or Absent Status    ${FANDevice_EM}    ${fan_bay2}    Absent
    #Check the Enclosure health status after efuse-on
    ${resp_after_efuseon}=    Fusion Api Get Enclosures
    ${health_after_efuseon}=    Get From Dictionary    ${resp_after_efuseon['members'][0]}    status
    Should Be Equal    ${health_after_efuseon}    Warning
    #Efuse-Off the fan bays and wait for the fan to return to OK status
    Efuse Fan Device   EFuseOff    ${fan_bay1}
    Wait For Device Status    ${FANDevice_EM}    ${fan_bay1}    OK
    Efuse Fan Device   EFuseOff    ${fan_bay2}
    Wait For Device Status    ${FANDevice_EM}    ${fan_bay2}    OK
    #Check the Enclosure health status after efuse-off
    ${resp_after_efuseoff}=    Fusion Api Get Enclosures
    ${health_after_efuseoff}=    Get From Dictionary    ${resp_after_efuseoff['members'][0]}    status
    Should Be Equal    ${health_after_efuseoff}    OK
    [Teardown]    Run Keyword If Test Failed    Run Keywords    Efuse Fan Device    EFuseOff    ${fan_bay1}    AND    Wait For Device Status    ${FANDevice_EM}    ${fan_bay1}    OK
    ...    AND   Efuse Fan Device    EFuseOff    ${fan_bay2}    AND    Wait For Device Status    ${FANDevice_EM}    ${fan_bay2}    OK
