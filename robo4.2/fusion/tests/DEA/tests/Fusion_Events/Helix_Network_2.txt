TC: 01 : Test Helix EM External Network port TransceiverFault on BAY1
    [Documentation]    Test Helix EM External Network port MauTransceiverFailureElinkPort and MauTransceiverFailureElinkPortCleared on BAY1
    [Tags]    EM    FLM3.00    Automated
    # Get enclsoureManager External Port Status and Health
    ${em1_status}=    Get AppliancePorts MAU Health    123    MAU
    Should Be Equal As Strings    ${em1_status}    OK
    ${em1_health}=    Get EnclosureManager Health    1
    Should Be Equal As Strings    ${em1_health}    OK
    ${em1_fault}=    Get AppliancePorts MAU Fault Status    123    MauTransceiverFailureElinkPort    MAU
    Should Be Equal As Strings    ${em1_fault}    OK
    ${status}=    Get SwitchPorts Fault Status    123    FaultMediumAttachmentUnit
    Should Be Equal As Strings    ${status}    OK
    ${last_fusion_event_id}=    get last fusion event ID
    #Set Fault ON
    Set EM MAU Faults    1    External    TransceiverFault    on
    Sleep    20
    ${diags_50}    ${diags_auditlog_file_path}=    Get Diags Audit Log 50 Recent Entries
    Set Test Variable    ${EM_Event_Logger}    ${diags_auditlog_file_path}
    #Verify Audit Log Alert Event    ${EM_EmMau1GbTransceiverElinkPort}    ${EM_MAU}    123    200
    Verify Fusion Event Sent By EM    ${EM_EmMauTransceiverFailureElinkPort}    ${last_fusion_event_id}    ${Enclosure_category}
    # Set Fault OFF
    Set EM MAU Faults    1    External    TransceiverFault    off
    Sleep    20
    ${diags_50}    ${diags_auditlog_file_path} =    Get Diags Audit Log 50 Recent Entries
    Set Test Variable    ${EM_Event_Logger}    ${diags_auditlog_file_path}
    Verify Fusion Event Sent By EM    ${EM_EmMauTransceiverFailureElinkPortCleared}    ${last_fusion_event_id}    ${Enclosure_category}
    [Teardown]    Run Keyword If Test Failed    Set EM MAU Faults    1    External    TransceiverFault    off
