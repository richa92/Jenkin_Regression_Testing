*** Keywords ***
Dialog Should Be Visible
    [Arguments]     ${name}    ${timeout}=2
    [Documentation]  Ensure that the dialog or section with title is visible
    Set Selenium Implicit Wait  ${timeout}
    Wait For Element Visible    xpath=//div[@id='hp-change-page-container' or contains(@class, 'hp-dialog') or contains(@class, 'hp-dialog-container')]//*[contains(text(), '${name}')]    timeout=${timeout}    fail_if_false=True
    Set Selenium Implicit Wait  ${SELENIUM_IMPLICIT_WAIT}

Dialog Should Not Be Visible
    [Arguments]     ${name}    ${timeout}=2
    [Documentation]  Ensure that the dialog or section with title is not visible
    Set Selenium Implicit Wait  ${timeout}
    Wait For Element Not Visible    xpath=//div[@id='hp-change-page-container' or contains(@class, 'hp-dialog') or contains(@class, 'hp-dialog-container')]//header//*[contains(text(), '${name}')]    timeout=${timeout}    fail_if_false=True
    Set Selenium Implicit Wait  ${SELENIUM_IMPLICIT_WAIT}

Dialog Button Should Be Visible
    [Arguments]     ${text}
    [Documentation]  Ensure that the dialog button with specified text is visible
    Element Should be Visible    xpath=//button[text()='${text}']

Dialog Element Should Be Visible
    [Arguments]     ${element}  ${timeout}=1
    [Documentation]  Ensure that the dialog Element is visible
    Set Selenium Implicit Wait  ${timeout}
    Wait For Element Visible   ${element}    timeout=${timeout}
    Element Should be Visible   ${element}
    Set Selenium Implicit Wait  ${SELENIUM_IMPLICIT_WAIT}

Dialog Element Should Not Be Visible
    [Arguments]     ${element}    ${timeout}=1
    [Documentation]  Ensure that the dialog Element is Not visible
    Set Selenium Implicit Wait  ${timeout}
    Wait For Element Not Visible   ${element}    timeout=${timeout}
    Element Should Not Be Visible   ${element}
    Set Selenium Implicit Wait  ${SELENIUM_IMPLICIT_WAIT}
