*** Keywords ***
Open Activity Side Bar
    [Documentation]    Click on Activity Icon and Ensure Activity Side bar is visible
    Click Element    css=div.hp-icon.hp-activity
    Wait Until Page Contains Element    css=div#hp-activity-flyout
    Element Should be Visible   xpath=//h1[text()='Activity']

Close Activity Side Bar
    [Documentation]    Ensure Activity Side Bar is not visible after clicking Icon twice
    Wait Until Page Contains Element    css=div.hp-icon.hp-activity
    Click Element    css=div.hp-icon.hp-activity
    Element Should Not Be Visible    xpath=//h1[text()='Activity']

Select Alert From Activity Page
    [Documentation]    Click Alert, if argument is not passed by default 1st alert will be selected
    [Arguments]    ${alert}=xpath=//table[@id='hp-activities']/tbody/tr[1]
    Click Element    ${alert}
    Page Should Contain Element    xpath=//table[@id='hp-activities']/tbody/tr[contains(@class, 'hp-selected')]
