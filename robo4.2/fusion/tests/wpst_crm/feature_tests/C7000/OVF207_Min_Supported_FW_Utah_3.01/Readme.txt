There are three main suites and 1 additional suite developed for this OVF-207 feature.
Suite_1 and Suite_3 has to be triggered in 3.10 appliance.
Suite_2 and presetup-for-suite3 has to be triggered in 3.00 appliance.
At the end of Suite_2 the appliance will be upgraded to 3.10 so do not use the same for presetup-for-suite3 suite.
Download the necessary SPP bundles and OV bin files into necessary folders.
