Get Provider URI
    [Documentation]    Get Provider URI by displayName.
    ...                displayName can be "Brocade Network Advisor", "Cisco", "Direct attach", "HP"
    ...                Example:  Get San Manager Provider Type URI  Brocade Network Advisor
    [Arguments]    ${displayName}
    ${resp} =    Fusion API Get Provider  param=?query=displayName eq '${displayName}'
	${count} =  Get From Dictionary  ${resp}  count
	Return From Keyword If  ${count}==0  /rest/provider_uri_${displayName}_not_found
	${uri} = 	Get From Dictionary		${resp['members'][0]}	uri
	[Return]	${uri}
